# Nykaa Product Analysis - EDA

## Objective
Analyzed Makeup, Healthcare, and Baby Care products from Nykaa 
to understand pricing trends, brand performance and customer popularity.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Web Scraping (BeautifulSoup / Requests)

## Key Analysis
- Category-wise brand comparison
- Price vs Rating analysis
- Discount patterns across categories
- Most popular brands by rating count

## Files
| File | Description |
|------|-------------|
| `nykaa_webscraping.ipynb` | Web scraping code |
| `nykaa_eda.ipynb` | Full EDA notebook |
| `nykaa_cleaned.csv` | Cleaned dataset |
| `nykaa_final.csv` | Final processed dataset |
